# Python Coffee Machine Simulator

A simple program that provides the option of 3 beverages which the user must select & pay for. The program will then check/deposit/refund the payment and deduct resources used to make the beverage providing there is enough to do so.

## How to Use

The file is written in standard Python so no extra modules are needed. The file can be copied/downloaded and run in any Python environment.
